iPark Backend API

This document describes the REST-API for use in the frontends (Website and App) of the iPark project. This API defines the data types and routes (e.g. actions and queries) required to interface the frontend with the backend. A special route for use in the IoT-Gateway is also described here.
Version: 1.0
